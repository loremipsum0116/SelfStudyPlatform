§ 01

집합의 기초 표기와 분류

문제 01~02

1.1 집합과 원소

집합(Set)은 명확한 기준에 따라 구별할 수 있는 대상들의 모임입니다. 집합은 보통 $A, B, C$ 같은 대문자로, 집합을 이루는 각 대상은 원소(Element)라 하며 $a, b, x$ 같은 소문자로 나타냅니다.

원소가 속한다 Membership
원소 $a$가 집합 $A$에 들어 있으면 $a \in A$ 로 씁니다.
원소가 속하지 않는다
원소 $a$가 집합 $A$에 들어 있지 않으면 $a \notin A$ 로 씁니다.
★ 가장 자주 틀리는 부분 $\in$ 은 원소와 집합 사이의 관계를 나타내는 기호입니다. 집합과 집합의 관계는 $\subseteq$ 같은 부분집합 기호를 써야 합니다.

1.2 집합의 표현 방법

원소나열법

집합에 속하는 원소를 중괄호 안에 직접 나열하는 방법입니다.

$A = \{1, 2, 3, 4, 5\}$

조건제시법

원소들이 만족하는 공통 조건을 이용해 집합을 나타내는 방법입니다. 이산수학 문제에서는 이 표현이 매우 자주 나옵니다.

$A = \{x \mid 1 \le x \le 5,\ x \in \mathbb{N}\}$

세로선 $\mid$ 앞은 원소의 형태, 뒤는 그 원소가 만족해야 하는 조건입니다.

1.3 자주 쓰는 수 체계 기호

$\mathbb{N}$
자연수 집합
$\mathbb{Z}$
정수 집합
$\mathbb{R}$
실수 집합

1.4 집합의 기수와 기본 분류

기수(Cardinality)는 집합의 원소 개수이며 $|A|$ 로 표기합니다.

유한집합 Finite Set
원소 개수가 유한한 집합입니다.
무한집합 Infinite Set
원소 개수가 끝없이 많은 집합입니다. 예: $\mathbb{N}$
공집합 Empty Set
원소가 하나도 없는 집합입니다. $\emptyset$ 또는 $\{\}$ 로 쓰며, $|\emptyset| = 0$ 입니다.
§ 02

원소와 집합의 포함 관계

문제 03~06

2.1 부분집합

집합 $A$의 모든 원소가 집합 $B$에도 속하면 $A$를 $B$의 부분집합이라고 하며 $A \subseteq B$ 로 씁니다.

$$A \subseteq B \iff \forall x(x \in A \to x \in B)$$
핵심 성질 모든 집합은 자기 자신의 부분집합입니다. 또한 공집합은 모든 집합의 부분집합입니다.

2.2 진부분집합

$A \subseteq B$ 이면서 동시에 $A \ne B$ 이면, $A$를 $B$의 진부분집합이라 하고 $A \subset B$ 로 씁니다. 즉, $B$에는 $A$에 없는 원소가 적어도 하나 이상 있습니다.

2.3 두 집합의 상등

두 집합이 정확히 같은 원소를 가지면 $A = B$ 입니다. 집합의 순서는 중요하지 않고, 같은 원소를 여러 번 써도 집합에서는 한 번만 취급합니다.

$\{1,2,3\} = \{3,2,1\}$ 이고, $\{1,1,2,2,3\} = \{1,2,3\}$ 입니다.

$A = B$ 를 보일 때는 보통 다음 두 가지를 모두 증명합니다.

$$A \subseteq B \quad \text{and} \quad B \subseteq A$$
§ 03

집합의 기본 연산

문제 07~12

집합의 연산은 챕터 2의 명제 논리와 연결해서 보면 훨씬 이해하기 쉽습니다. 합집합은 OR, 교집합은 AND, 여집합은 NOT과 대응합니다.

3.1 합집합

집합 $A$ 또는 집합 $B$에 속하는 원소들의 집합입니다.

$$A \cup B = \{x \mid x \in A \lor x \in B\}$$

3.2 교집합

집합 $A$와 $B$에 공통으로 들어 있는 원소들의 집합입니다.

$$A \cap B = \{x \mid x \in A \land x \in B\}$$
서로소 두 집합의 공통 원소가 하나도 없으면 $A \cap B = \emptyset$ 이고, 이때 두 집합을 서로소라고 합니다.

3.3 차집합

$A$에는 속하지만 $B$에는 속하지 않는 원소들의 집합입니다.

$$A - B = \{x \mid x \in A \land x \notin B\}$$
필수 변환 차집합은 계산할 때 자주 $A - B = A \cap B^c$ 형태로 바꾸어 사용합니다.

3.4 여집합

전체집합 $U$가 주어져 있을 때, $A$에 속하지 않는 원소들의 집합을 $A$의 여집합이라고 합니다.

$$A^c = U - A = \{x \mid x \in U \land x \notin A\}$$

여집합은 전체집합이 무엇인지가 먼저 정해져야만 의미가 있습니다.

3.5 대칭차집합

두 집합 중 정확히 한쪽에만 속하는 원소들의 집합입니다. 즉, 겹치는 부분은 빼고 남는 부분만 모읍니다.

$$A \oplus B = (A-B) \cup (B-A) = (A \cup B) - (A \cap B)$$
§ 04

포함-배제의 원리와 기수 계산

문제 13~16, 20~27

집합 문제에서 개수를 묻는 문장제는 대부분 포함-배제의 원리로 풀립니다. 핵심은 겹친 부분을 중복해서 세지 않도록 조정하는 것입니다.

4.1 두 집합의 포함-배제 원리

$$|A \cup B| = |A| + |B| - |A \cap B|$$

$|A| + |B|$를 그대로 더하면 교집합 부분이 두 번 더해지므로, $|A \cap B|$를 한 번 빼 주어야 합니다.

4.2 세 집합의 포함-배제 원리

$$|A \cup B \cup C| = |A| + |B| + |C| - |A \cap B| - |B \cap C| - |C \cap A| + |A \cap B \cap C|$$

세 집합의 경우에는 두 집합씩 겹치는 부분을 먼저 빼고, 그러면 세 집합이 모두 겹치는 부분이 너무 많이 빠지므로 마지막에 한 번 다시 더합니다.

4.3 자주 쓰는 기수 공식

차집합의 원소 개수
$|A-B| = |A| - |A \cap B|$
여집합의 원소 개수
$|A^c| = |U| - |A|$
대칭차집합의 원소 개수
$|A \oplus B| = |A \cup B| - |A \cap B|$
문장제에서 먼저 해야 할 일은 조건을 집합으로 바꾸는 것입니다. 예를 들어 전체 학생 집합을 $U$, 축구를 하는 학생 집합을 $A$, 농구를 하는 학생 집합을 $B$로 두면, “둘 다 하는 학생 수”는 $|A \cap B|$, “적어도 하나는 하는 학생 수”는 $|A \cup B|$가 됩니다.
§ 05

곱집합과 멱집합

문제 17~19

5.1 곱집합

집합 $A$의 원소를 첫 번째 성분으로, 집합 $B$의 원소를 두 번째 성분으로 하는 순서쌍들의 집합을 곱집합이라고 합니다.

$$A \times B = \{(a,b) \mid a \in A,\ b \in B\}$$
주의 순서쌍은 순서가 중요하므로 일반적으로 $(a,b) \ne (b,a)$ 입니다. 따라서 보통 $A \times B \ne B \times A$ 입니다.
$$|A \times B| = |A|\cdot|B|$$

5.2 멱집합

집합 $A$의 모든 부분집합을 원소로 갖는 집합을 멱집합이라 하고 $P(A)$ 또는 $2^A$ 로 표기합니다.

$A = \{1,2\}$ 이면 부분집합은 $\emptyset, \{1\}, \{2\}, \{1,2\}$ 이므로 $$P(A)=\{\emptyset, \{1\}, \{2\}, \{1,2\}\}$$

원소가 $n$개인 집합의 멱집합 원소 개수는 다음과 같습니다.

$$|P(A)| = 2^{|A|}$$

이 공식은 각 원소를 “부분집합에 넣을지 말지” 두 가지 선택으로 생각하면 쉽게 이해할 수 있습니다.

§ 06

집합의 대수 법칙

문제 28~31

집합식의 간단화는 명제 논리의 동치 법칙과 거의 같은 구조입니다. 식을 줄이거나 같음을 증명할 때 자주 사용합니다.

6.1 교환법칙

$$A \cup B = B \cup A \qquad A \cap B = B \cap A$$

6.2 결합법칙

$$(A \cup B) \cup C = A \cup (B \cup C)$$ $$(A \cap B) \cap C = A \cap (B \cap C)$$

6.3 분배법칙

$$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$ $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

6.4 드 모르간의 법칙

$$\overline{A \cup B} = \bar{A} \cap \bar{B}$$ $$\overline{A \cap B} = \bar{A} \cup \bar{B}$$

전체에 걸린 여집합을 풀면, 합집합과 교집합 기호가 서로 뒤바뀝니다.

6.5 흡수법칙

$$A \cup (A \cap B) = A$$ $$A \cap (A \cup B) = A$$

6.6 차집합 변환

$$A-B = A \cap \bar{B}$$
실전 팁 집합식 간단화 문제에서는 차집합 기호를 먼저 $A \cap \bar{B}$ 형태로 바꾼 뒤, 드 모르간 법칙과 분배법칙을 적용하면 훨씬 정리가 쉬워집니다.
