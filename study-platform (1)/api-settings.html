Google Gemini API

Gemini API 키를 등록하면 대시보드에서 AI 어시스턴트를 사용할 수 있습니다.
등록된 키는 계정에 저장되어 다시 로그인해도 유지됩니다.

API 키 발급 방법

  1. Google AI Studio에 접속합니다.
  2. Google 계정으로 로그인합니다.
  3. Create API Key 버튼을 클릭합니다.
  4. 생성된 키(AIzaSy...)를 복사합니다.
  5. 위 입력란에 붙여넣고 저장을 누릅니다.

⚠️ API 키는 개인 정보입니다. 타인에게 공유하지 마세요.
무료 등급에서도 분당 일정 횟수의 요청이 가능합니다.
